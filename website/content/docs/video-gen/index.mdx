---
title: 视频生成 API 参考
description: 基于 Seedance 系列模型的异步视频生成 API，支持文生视频、图生视频（首帧/首尾帧/参考图）。
---

视频生成为**异步接口**，分两步完成：

1. `POST /contents/generations/tasks` — 创建任务，获得任务 ID
2. `GET /contents/generations/tasks/{id}` — 轮询状态，`status: succeeded` 后从 `content.video_url` 下载 mp4

## 创建任务

`POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks`

### 请求参数

#### model `string` (必需)

模型 ID，见上方模型表。

#### content `object[]` (必需)

输入内容，支持文本和图片组合。

**文本**

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `type` | string | ✓ | 固定为 `"text"` |
| `text` | string | ✓ | 提示词（中文 ≤ 500 字，英文 ≤ 1000 词） |

**图片**

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `type` | string | ✓ | 固定为 `"image_url"` |
| `image_url.url` | string | ✓ | 图片 URL 或 Base64 |
| `role` | string | 条件必填 | 多图时必填，见下表 |

**`role` 取值规则：**

| 场景 | 图片数量 | role |
|------|---------|------|
| 图生视频（首帧） | 1 张 | `first_frame`（或不填） |
| 图生视频（首尾帧） | 2 张 | 首帧 `first_frame`，尾帧 `last_frame` |
| 图生视频（参考图，仅 lite i2v） | 1–4 张 | 全部填 `reference_image` |

图片要求：jpeg/png/webp/bmp/tiff/gif；宽高比 (0.4, 2.5)；边长 300–6000 px；< 30 MB。

#### resolution `string` (可选)

可选：`480p` / `720p` / `1080p`（参考图场景不支持）

#### ratio `string` (可选)

可选：`16:9` / `4:3` / `1:1` / `3:4` / `9:16` / `21:9` / `adaptive`

`adaptive`：图生视频（首帧/首尾帧）根据首帧图片比例自动适配；文生视频仅 Seedance 1.5 pro 支持。

#### duration `integer` (可选，默认 `5`)

视频时长（秒）。Seedance 1.0 系列 [2, 12]，Seedance 1.5 pro [4, 12]（或 `-1` 由模型自动选择）。与 `frames` 互斥。

#### frames `integer` (可选，Seedance 1.5 pro 不支持)

精确控制帧数（24fps）。取值范围：[29, 289] 内满足 `25 + 4n` 的整数。与 `duration` 互斥。

#### generate_audio `boolean` (可选，默认 `true`，仅 Seedance 1.5 pro)

是否生成同步音频（人声、音效、背景音乐）。

#### return_last_frame `boolean` (可选，默认 `false`)

是否在响应中返回视频尾帧图像（png）。

#### seed `integer` (可选，默认 `-1`)

随机种子，范围 [-1, 2³²-1]。相同 seed 生成类似结果，但不保证完全一致。

#### camera_fixed `boolean` (可选，默认 `false`)

是否固定摄像头。参考图场景不支持。

#### watermark `boolean` (可选，默认 `false`)

是否添加水印。

### 请求示例

**文生视频**

```bash
curl -X POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks \
  -H "Authorization: Bearer $ARK_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1-5-pro-251215",
    "content": [
      { "type": "text", "text": "小猫对着镜头打哈欠" }
    ],
    "resolution": "720p",
    "ratio": "16:9",
    "duration": 5,
    "watermark": false
  }'
```

**图生视频（首帧）**

```bash
curl -X POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks \
  -H "Authorization: Bearer $ARK_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1-5-pro-251215",
    "content": [
      {
        "type": "image_url",
        "image_url": { "url": "https://example.com/first-frame.jpg" },
        "role": "first_frame"
      },
      { "type": "text", "text": "小猫缓缓抬起头，望向远方" }
    ],
    "duration": 5
  }'
```

### 响应

```json
{
  "id": "cgt-2025****-****"
}
```

保存任务 ID，用于后续查询。

## 查询任务

`GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks/{id}`

仅支持查询最近 7 天的任务。

### 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 任务 ID |
| `model` | string | 使用的模型 ID |
| `status` | string | 任务状态，见下表 |
| `content.video_url` | string | 视频下载地址（mp4），有效期 24 小时 |
| `content.last_frame_url` | string | 视频尾帧，有效期 24 小时（需 `return_last_frame: true`） |
| `usage.completion_tokens` | integer | 输出视频消耗的 token 数 |
| `created_at` | integer | 任务创建时间（Unix 时间戳） |
| `updated_at` | integer | 状态最后更新时间 |
| `resolution` | string | 生成视频分辨率 |
| `ratio` | string | 生成视频宽高比 |
| `duration` | integer | 生成视频时长（秒） |
| `framespersecond` | integer | 帧率（固定 24） |

**status 枚举值：**

| 值 | 说明 |
|----|------|
| `queued` | 排队中 |
| `running` | 生成中 |
| `succeeded` | 已完成 |
| `failed` | 已失败 |
| `cancelled` | 已取消（仅排队中可取消） |
| `expired` | 已超时 |

### 响应示例

```json
{
  "id": "cgt-2025****-****",
  "model": "doubao-seedance-1-5-pro-251215",
  "status": "succeeded",
  "content": {
    "video_url": "https://ark-content-generation-cn-beijing.tos-cn-beijing.volces.com/****"
  },
  "usage": {
    "completion_tokens": 246840,
    "total_tokens": 246840
  },
  "created_at": 1765510475,
  "updated_at": 1765510559,
  "seed": 58944,
  "resolution": "1080p",
  "ratio": "16:9",
  "duration": 5,
  "framespersecond": 24,
  "generate_audio": true
}
```

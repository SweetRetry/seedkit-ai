本站是豆包大模型 API 的参考文档，基于官方文档整理，提供更简洁的阅读体验。

## API 文档

| API | 说明 |
|-----|------|
| [Chat API](/docs/text-gen) | 对话补全，支持流式输出、多模态、Function Calling、结构化输出、深度思考 |
| [Responses API](/docs/text-gen-responses) | 新一代对话接口，支持多轮会话管理、联网搜索 |
| [图像生成](/docs/image-gen) | 基于 Seedream 系列模型，支持文生图、图生图、多图融合 |
| [视频生成](/docs/video-gen) | 基于 Seedance 系列模型，支持文生视频、图生视频 |

## 鉴权

所有 API 均通过 HTTP Header 传递 API Key：

```
Authorization: Bearer $ARK_API_KEY
```

API Key 可在[火山引擎控制台](https://console.volcengine.com/ark)获取。
